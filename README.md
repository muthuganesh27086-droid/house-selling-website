# 🏠 Home Selling Website

A modern and responsive Home Selling Website developed using HTML and CSS.  
This project was created as part of my internship learning experience to strengthen my front-end development skills.

---

## 📌 Project Overview

The Home Selling Website is designed to showcase properties available for sale in a clean and user-friendly layout.  
It includes structured sections for property listings, navigation, and contact information.

---

## 🚀 Features

- Responsive and modern UI design  
- Property listing sections  
- Navigation menu  
- Contact section  
- Clean layout and structured design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  

---

## 📚 What I Learned

- Structuring web pages using semantic HTML  
- Styling and layout design using CSS  
- Creating responsive web designs  
- Improving front-end development fundamentals

This project was built to apply the concepts learned during my internship training and to gain practical experience in designing real-world web interfaces.
